NIHANTH COMPANY HCM - Enterprise Portal

A professional, high-fidelity HCM solution inspired by Workday, built for scalability and ease of use.

Technical Stack





Framework: React 18+ (Vite-powered)



Styling: Tailwind CSS for enterprise-grade layouts



Icons: Lucide-React



Visualization: Recharts for HCM analytics



Deployment: Optimized for Vercel

Project Structure





/src: Contains core application logic and UI components.



index.html: Main entry point.



package.json: Dependency management and build scripts.



vercel.json: Deployment configuration for Vercel.

Design Philosophy

The UI follows a strict enterprise design system characterized by:





Clarity: High-contrast typography (Inter) and clear visual hierarchy.



Efficiency: Quick actions and densified data tables for power users.



Consistency: Unified color palette (#003366 primary) across all modules.

Getting Started





Install dependencies: npm install



Run development server: npm run dev



Build for production: npm run build
